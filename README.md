# Gexton Sales Data Analysis Dashboard

A professional Tkinter/CustomTkinter dashboard built for the Python Programming Task #08 during the Gexton Summer Internship Program[cite: 1].

## Features
- Modern, high-contrast dark theme layout using CustomTkinter.
- Task 1: Comprehensive Sales Data Overview and metrics compilation[cite: 1].
- Task 2: Embedded Matplotlib charts showing trends and regional distribution[cite: 1].
- Dynamic management report generator including feature correlation analysis[cite: 1].
- Memory leak and background thread crash prevention on application termination.

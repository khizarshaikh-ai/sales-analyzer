# Gexton Sales Data Analysis Dashboard

A professional Tkinter/CustomTkinter dashboard built for the Python Programming Task #08 during the Gexton Summer Internship Program.

## Features
- Modern, high-contrast dark theme layout using CustomTkinter.
- Task 1: Comprehensive Sales Data Overview and metrics compilation.
- Task 2: Embedded Matplotlib charts showing trends and regional distribution.
- Dynamic management report generator including feature correlation analysis.
- Memory leak and background thread crash prevention on application termination.
